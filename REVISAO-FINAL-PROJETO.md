## 🛠️ Customização Rápida (< 30 min)

Para criar nova landing page:

1. **Copy projeto** (ou branch)
2. **Editar 5 arquivos:**
   - `config/content.ts` (texto)
   - `config/theme.ts` (cores)
   - `config/layouts.ts` (layouts)
   - `public/logos/` (logo)
   - `public/images/` (mídia)

3. **Deploy:**
   ```bash
   npm run build
   # Deploy no Vercel/Netlify em 1 clique
   ```
